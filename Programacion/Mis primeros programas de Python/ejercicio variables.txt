1.Las varibles sirven para designar un valor, lo cual aplicamos en este ejercicio para designarle un valor a la poblacion y nombre_nacion.
Siendo esta capacidad de almacenar informacion en las variables, un factor clave para el desarrollo de programas mas complejos.

2.Cuando declaramos una variable tenemos, en primer lugar, el identificador que nombra la variable, seguido de un =, y por último el valor asignado a la variable.

Por ejemplo, el siguiente código muestra la creación de una variable y la asignación de un valor:

```
nombre_nacion = "España"
```

Y para mostrar estas varible se usa el comando print.

Por ejemplo, el siguiente codigo mustra como se mostraria:

```
print("nombre_nacion")
```

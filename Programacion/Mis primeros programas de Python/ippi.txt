Instanciar un objeto significa crear una variable que guarda un dato, como un número o texto. Por ejemplo, cuando usamos int(10) creamos un objeto entero con el valor 10. Cuando usamos float(3.14) creamos un objeto de tipo número con decimales. El módulo math nos da funciones listas para usar, como sqrt(), que calcula la raíz cuadrada de un número.
2y3:
#Importo la libreria math
import math
#Designo el valor 10 a numero_entero convirtiendolo en un valor entero con el comando int
numero_entero = int(10)
#Designo una variable con el valor 3.14 con el comando float
numero_pi = float(3.14)
#Creo un objeto str con el valor "Hola mundo" y lo asigno a la variable saludo
saludo = "¡Hola mundo!"
#Calculo la raiz cuadrada de 16 con sqrt y le asigno el valor resultado a raiz_cuadrada
raiz_cuadrada = math.sqrt(16)

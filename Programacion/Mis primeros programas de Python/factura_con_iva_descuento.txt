de aplicarle un descuento de 10€ si el precio del producto tras aplicar

#Inserto las variables pertinentes para la creacion del programa
#Le pregunto al cliente Su nombre y el precio del producto a calcular, ademas de convertir el precio dado por la persona en decimal.
nombre_cliente = input("¿Como te llamas?: ")

precio_bruto = int(input("¿Cuanto cuesta tu producto?: "))
#Hago que el IVA sea decimal con el comando float, ademas de una constante poniendola en mayusculas, para evitar que se cambie su valor en cualquier momento.
´´´
IVA = float(0.21)
´´´
#La variable descuento tambien la hago constante para evitar problemas, ademas de usar el comando int para que sea un numero entero.
´´´
DESCUENTO = int(10)
´´´
´´´
descuento_aplicado = 0

total_a_pagar = 0
´´´
#Realizo las operaciones pertinentes para calcular el IVA_APLICADO y el subtotal_con_iva.
´´´
IVA_APLICADO = precio_bruto * IVA
´´´
´´´
subtotal_con_iva = precio_bruto + IVA_APLICADO
´´´
#Utilizo los comandos el y elif para calcular si se le debe aplicar el descuento a subtotal_con_iva, dependiendo de si es igual o mas que 50 o menos, ademas de añadir lo descontado en caso de ello a la variable descuento_aplicado
´´´
if subtotal_con_iva >= 50:
    total_a_pagar = subtotal_con_iva - DESCUENTO
    descuento_aplicado = 10
elif subtotal_con_iva < 50:
    total_a_pagar = subtotal_con_iva
    descuento_aplicado = 0
´´´
#Muestro todos los datos finales.
´´´
print(nombre_cliente)
print(precio_bruto)
print(IVA_APLICADO)
print(descuento_aplicado)
print(total_a_pagar)
´´´

# Rúbrica: Gestión de caballos y cuadras en Python

## Introducción y contextualización 
En este ejercicio se plantea un problema relacionado con la gestion del transporte y vivienda de caballos en cuadras.  
Se pretende calcular cuántas cuadras son necesarias para alojar a un número determinado de caballos, considerando que cada cuadra tiene una capacidad limitada de 3 como maximo.  
Este escenario simula la planificación de recursos y logística dentro de un dilema de gestion que podria llegar a tener un ranchero, mostrando cómo la programación puede ayudar a tomar decisiones eficientes en situaciones de gestión sin necesidad de complicarse.

---

## Desarrollo técnico correcto y preciso 
El código implementa correctamente los conceptos de **variables, constantes y operaciones básicas**:

```python
import math  # Importamos el módulo math para usar ceil

numero_caballos = int(input("¿Cuantos caballos hay disponibles?: "))

CABALLOS_POR_CUADRA = 3  # Constante que indica la capacidad de cada cuadra

# Calculamos el número de cuadras necesarias, redondeando hacia arriba
numero_cuadras = math.ceil(numero_caballos / CABALLOS_POR_CUADRA)

# Mostramos los resultados
print("Tienes", numero_caballos, "caballos")
print("Necesitarías", numero_cuadras, "cuadras para el transporte")
Se usan variables para almacenar la cantidad de caballos y cuadras.

Se usa una constante (CABALLOS_POR_CUADRA) para indicar la capacidad fija de cada cuadra.

Se realiza una operación matemática básica (división) y se redondea hacia arriba con math.ceil para garantizar que todos los caballos tengan espacio.

Aplicación práctica:
Supongamos que el jugador tiene 10 caballos:
Tienes 10 caballos
Necesitarías 4 cuadras para el transporte
Explicación:

Cada cuadra puede alojar 3 caballos.

10 ÷ 3 = 3.33 → con math.ceil, se redondea a 4 cuadras.
Este ejemplo demuestra claramente cómo el código resuelve un problema práctico de planificación en un juego de estrategia.
